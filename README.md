Getting Started
--------

### prerequisites
- Python ≥ *3.5*

### Installation
```bash
sudo cp zweihbot.service /etc/systemd/system
sudo systemctl daemon-reload

sudo systemctl enable hyeonbot
sudo systemctl start hyeonbot
```
